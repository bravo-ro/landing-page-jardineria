# Huerto Feliz - Landing Page

Este proyecto es una Landing Page estática y responsiva desarrollada para una empresa ficticia de servicios de jardinería llamada "Huerto Feliz". Fue creado como proyecto académico para demostrar el dominio en la estructuración semántica web y el diseño moderno sin depender de frameworks externos.

## Características Principales

* **Diseño Responsivo:** La página se adapta a cualquier tamaño de pantalla, desde teléfonos móviles hasta monitores de escritorio.
* **Animaciones CSS:** Incluye un Hero Header con transición automática de imágenes de fondo (carrusel infinito) y efectos visuales al interactuar con las tarjetas de servicios.
* **Accesibilidad:** Uso de tipografías legibles, contrastes adecuados y estructura de etiquetas HTML5 semánticas.
* **JavaScript:** El uso de JS se limitó estrictamente a la funcionalidad del menú de navegación colapsable en móviles y a la prevención del envío por defecto del formulario de contacto.

## Tecnologías Utilizadas

* **HTML5** (Estructura semántica)
* **CSS3** (Estilos, Grid/Flexbox, Animaciones `@keyframes`, Media Queries)
* **JavaScript** (Interacción básica)
